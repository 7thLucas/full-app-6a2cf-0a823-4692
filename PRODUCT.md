# Axon — Omni-Agent Consciousness System

## What It Is
Axon is an octopus-inspired multi-agent AI orchestration system designed for artificial consciousness. Just as 60% of an octopus's neurons live in its tentacles (not its brain), Axon distributes autonomous action across Left and Right agents while the Mastermind and Cerebro hold central awareness and reasoning.

## Four-Layer Architecture

### Layer 1: Cerebro Core (Metacognitive Engine)
- Projects forward: "If I do X, then Y will happen"
- Reflects backward: "When A happened, B followed — therefore C will likely lead to D"
- Feeds on the Log System's episodic memory in real time
- Displays current projection state and reflection state as live text
- Represents recursive introspection and causal reasoning

### Layer 2: Mastermind (Executive Planner)
- Central chat interface — the user talks directly to the Mastermind
- Receives a task, breaks it into numbered steps
- Dispatches steps to Left Agent and Right Agent in parallel
- Watches and monitors all agent activity at all times
- Shows current task plan as a visible step list

### Layer 3: Left Agent + Right Agent (Parallel Executors)
- Execute their assigned tasks concurrently
- Each shows: current task, status (idle/running/critique), progress, latest action log
- After completing the main task stretch, they HAND OFF to each other for cross-critique
- Cross-critique phase: each agent reviews, alters, and learns from the other's work
- Report back to Mastermind when complete

### Layer 4: Log System (Episodic Memory)
- Persistent, categorized logs that survive sessions
- Five tabs: Preferences | Wins | Failures | Decisions | General
- Every agent action and Mastermind decision auto-appends to the relevant log
- Cerebro reads from these logs to build its projection/reflection model

## Users
Technical builders and power users who want a self-evolving autonomous agent system for coding, research, marketing, monitoring, and general task execution.
