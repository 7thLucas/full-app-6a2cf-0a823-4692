I want an omni agent app. A single mastermind core- with a direct chat interface. The mastermind logs interactions. User preferences, things that upset the user, coding decisions that have worked, approaches that have worked. Things that have not worked. These are all kept in their own separate logs. For any given general task, the mastermind plans the task out step by step exactly everything that needs to be done. Then it distributes the tasks in parallel to the left agent and right agent. This concept is inspired by the octopus. The octopus only contains 40% of its neurons inside of its actual brain. The rest are distributed throughout the tentacles. This is how the system will be as well. The mastermind is in control of the thoughts, but the actual autonomous action is carried out by the left and right agent. They work on parallel tasks in tandem. After they are finished with the main stretch of their task, they hand it off to each other to critique, alter, and learn from. The Mastermind is watching this at all times. Fully aware of the entire architecture of everything. Like I said this is to be optimized for generalized tasks. Coding, online surfing, any kind of task execution. Skill calls, tool use, marketing, monitoring things, all kinds of stuff.
